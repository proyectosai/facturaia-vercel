# Código de conducta

## Nuestro compromiso

FacturaIA quiere ser un proyecto útil, abierto y técnicamente serio. Para ello, el espacio de colaboración debe seguir siendo respetuoso, claro y productivo.

Se espera de todas las personas participantes:

- respeto en el trato
- crítica técnica centrada en el trabajo, no en la persona
- apertura a revisar ideas con argumentos
- cuidado con el lenguaje, especialmente en discusiones públicas

## Conductas esperadas

- hacer preguntas con buena fe
- aceptar correcciones técnicas razonables
- aportar contexto suficiente cuando se reporta un problema
- respetar el tiempo de revisión de otras personas
- mantener el foco en mejorar el proyecto

## Conductas no aceptables

- ataques personales
- lenguaje insultante o discriminatorio
- acoso o comportamiento intimidatorio
- publicar información privada de terceros
- sabotear discusiones o revisiones de forma deliberada

## Ámbito

Este código de conducta aplica a:

- issues
- pull requests
- discusiones públicas del repositorio
- documentación y revisiones relacionadas con FacturaIA

## Aplicación

Si una interacción rompe estas normas, la persona mantenedora podrá:

- pedir corrección del tono o del comportamiento
- cerrar una conversación improductiva
- rechazar una contribución
- limitar participación en el repositorio si fuese necesario

## Interpretación

La prioridad es proteger un entorno de trabajo útil para quienes construyen y revisan el proyecto.

Si tienes dudas sobre cómo enfocar una interacción, prioriza claridad, respeto y contexto técnico.
